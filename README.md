# Лабораторные работы по алгоритмам и структурам данных

Репозиторий содержит консольное приложение для выполнения пяти лабораторных работ по алгоритмам и структурам данных на языке C.

## Структура проекта

```
├── src/
│   ├── 1/              # Лабораторная работа 1
│   │   ├── AL1.h       # Заголовочный файл для ЛР1
│   │   └── AL1.c       # Реализация ЛР1
│   ├── 2/              # Лабораторная работа 2
│   │   ├── AL2.h
│   │   └── AL2.c
│   ├── 3/              # Лабораторная работа 3
│   │   ├── AL3.h
│   │   └── AL3.c
│   ├── 4/              # Лабораторная работа 4
│   │   ├── AL4.h
│   │   └── AL4.c
│   │   └── input.txt   # Входные данные для ЛР4
│   ├── 5/              # Лабораторная работа 5
│   │   ├── AL5.h
│   │   └── AL5.c
│   │   └── graph.txt   # Входные данные для ЛР5
│   └── main.c          # Основная программа
└── README.md
```

## Описание лабораторных работ

### 1. Основы работы с консольным приложением и синтаксический разбор текста
- **Цель**: Работа с файловым вводом-выводом и анализ текста
- **Запуск**: Через основное меню, требует указания имени файла
- **Функциональность**: Обработка текстовых файлов, синтаксический анализ

### 2. Сортировка одномерных массивов
- **Цель**: Реализация различных алгоритмов сортировки
- **Функциональность**: Сортировка массивов разными методами

### 3. Линейные списки
- **Цель**: Работа с линейными односвязными и двусвязными списками
- **Функциональность**: Операции вставки, удаления, поиска в списках

### 4. Двоичные деревья поиска
- **Цель**: Реализация и работа с двоичными деревьями поиска
- **Входные данные**: `src/AL4/input.txt`
- **Функциональность**: Построение деревьев, обходы, балансировка

### 5. Поиск кратчайших путей в ориентированном взвешенном графе
- **Цель**: Алгоритмы поиска кратчайших путей в графах
- **Входные данные**: `src/AL5/graph.txt`
- **Функциональность**: Реализация алгоритмов Дейкстры или Флойда-Уоршелла

## Сборка и запуск

### Компиляция

```bash
# Компиляция всех модулей
gcc -o main src/main.c src/1/AL1.c src/2/AL2.c src/3/AL3.c src/4/AL4.c src/5/AL5.c

# Или поэтапная компиляция
gcc -c src/1/AL1.c -o AL1.o
gcc -c src/2/AL2.c -o AL2.o
gcc -c src/3/AL3.c -o AL3.o
gcc -c src/4/AL4.c -o AL4.o
gcc -c src/5/AL5.c -o AL5.o
gcc -c src/main.c -o main.o
gcc main.o AL1.o AL2.o AL3.o AL4.o AL5.o -o main
```

### Запуск

```bash
./main
```

## Использование

1. При запуске программы отображается меню с пятью вариантами лабораторных работ
2. Выберите номер лабораторной работы (1-5)
3. Для некоторых работ потребуется указать входные данные
4. После выполнения работы можно вернуться в главное меню

## Особенности реализации

- **Кроссплатформенность**: Использует условную компиляцию для очистки экрана (Windows/Linux)
- **Модульность**: Каждая лабораторная работа реализована в отдельном модуле
- **Интерактивность**: Возможность последовательного выполнения нескольких работ

## Требования

- Компилятор C (GCC, Clang, MSVC)
- Стандартная библиотека C
- Операционная система: Windows, Linux или macOS

## Примечания

- Лабораторная работа 1 запускается как внешняя программа с передачей имени файла
- Для работ 4 и 5 используются предопределенные файлы с входными данными
- Программа поддерживает рекурсивный возврат в главное меню
